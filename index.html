Recipe Book

Technologies used : Angular , FireBase
Link : Recipe Book

Recipe Book is a web application built using AngularJS that serves as a platform for users to discover, create, and learn new recipes. The website leverages Firebase as its backend database to store a collection of recipes. Users can browse through the available recipes, accessing detailed instructions and ingredient lists for each dish.

One of the key features of Recipe Book is its integrated shopping list functionality. When users find a recipe they want to try, they can add all the necessary ingredients to their shopping list with just a click. This simplifies the process of gathering ingredients for cooking, making meal preparation more efficient and organized.

Overall, Recipe Book provides a user-friendly interface for recipe discovery and management, enhancing the cooking experience for users by combining the power of AngularJS with the flexibility of Firebase for data storage and retrieval.

Ecommerce API

Technologies used : SpringBoot , MySQL
Link : Ecommerce API

The Ecommerce-API is a powerful application programming interface (API) developed using Spring Security for robust authentication and authorization functionalities. It serves as a comprehensive platform for managing users, administrators, and product transactions within an e-commerce ecosystem.

With Spring Security integration, the API ensures secure access to its features, allowing users and administrators to authenticate themselves securely. User management capabilities enable the creation of both regular users and administrators, each with their respective privileges and permissions.

The API facilitates product management by categorizing items, providing a structured approach to organizing the inventory. Users can browse through various product categories, add items to their carts, and proceed with purchases seamlessly.

Administrators have enhanced functionalities, including the ability to monitor user activities, track transactions, and manage product listings. They can also add new products to the inventory, ensuring that the platform stays updated with the latest offerings.

Overall, the Ecommerce-API streamlines e-commerce operations, ensuring a secure, efficient, and user-friendly experience for both customers and administrators alike. Its integration with Spring Security enhances data protection and access control, making it a reliable solution for managing online transactions and user interactions.
